# NFT Marketplace

A decentralized NFT marketplace built with Ethereum (Solidity, Hardhat) and React (Vite, Ethers.js, Tailwind/Vanilla CSS).

## Setup
1. Run `npm install` in the root directory.
2. Run `npm install` in the `frontend` directory.
3. Start a local hardhat node: `npx hardhat node`
4. Deploy contracts: `npx hardhat run scripts/deploy.js --network localhost`
5. Start frontend: `cd frontend && npm run dev`
